## 概要

---

Orthrosを利用したユーザーの一覧を確認出来ます（参照のみ）

## 手順

---
>[!NOTE]
>[Orthros マイページ](https://ui.orthros.gakunin.nii.ac.jp)にログインの上確認をお願いします。  
>※テスト環境の場合は(https://ui-stg.orthros.gakunin.nii.ac.jp)  となります。  
>「SP管理」メニューが表示されるためには、対象アカウントに対してSP管理権限の付与が必要となります。  
>SP管理権限の付与については申請が必要となります。  
>申請方法はマイページよりお問い合わせください。（お問い合わせ方法は[利用マニュアル](/docs/利用マニュアル/README.md)の「[お問い合わせ方法](/docs/利用マニュアル/README.md#お問い合わせについて)」をご参照ください。）

**1\. ユーザー名のプルダウンから「SP管理」を選択します。**

<img src="/images/このSPを利用中のユーザー-1.png" width="54%">

  

**2\. 該当SPの「管理」ボタンをクリックします。**

<img src="/images/このSPを利用中のユーザー-2.png" width="51.4%">

  

**3\. 利用ユーザーの一覧が表示されます。**

<img src="/images/このSPを利用中のユーザー-3.png" width="67.1%">

  

表3 項目一覧

| 項目 | 説明 |
| --- | --- |
| 所属機関 | 利用ユーザーが所属している機関<br>⚠️**本内容は将来実装予定の機能となっております。（2023/08/01）** |
| 氏名 | 利用ユーザーの氏名 |
| メールアドレス | 利用ユーザーのメールアドレス |
| IAL | 利用ユーザーがマイページに最終アクセスした際の本人確認レベル（実装予定） |
| 最終同意日時 | 利用ユーザーがSPに対して最後に属性送信に同意した日時 |
| 最終ログイン日時 | 利用ユーザーがSPに対して最後にログインした日時 |
