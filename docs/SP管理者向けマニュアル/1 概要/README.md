## 1\. 対象者
本マニュアルは、Orthrosと接続するSPの管理者（以降SP管理者）を対象としています

## 2\. SP管理者への提供機能について
OrthrosではSP管理者向けに以下の機能を提供しています

  

表 2 提供機能一覧

| 機能 | 説明 |
| --- | --- |
| [管理対象のSP](/docs/SP管理者向けマニュアル/3%20SP管理者で利用する機能について/1%20管理対象のSP.md) | 自身がSP管理者となっているSPの一覧を確認出来ます |
| [このSPを利用中のユーザー](/docs/SP管理者向けマニュアル/3%20SP管理者で利用する機能について/2%20このSPを利用中のユーザー.md) | Orthrosを利用したユーザーの一覧を確認出来ます（参照のみ） |
| [要求する保証レベル](/docs/SP管理者向けマニュアル/3%20SP管理者で利用する機能について/3%20要求する保証レベル.md) | OrtrosSPがユーザーに対して要求する保証レベル（本人確認レベル、認証プロセスの保証レベル）を設定します<br>この機能により、SPが要求する保証レベルのユーザーのみアクセス出来ることを保証します<br>⚠️**本内容は将来実装予定の機能となっております。（2023/08/01）** |
| [送出属性の設定](/docs/SP管理者向けマニュアル/3%20SP管理者で利用する機能について/4%20送出属性の設定.md) | OrthrosからSPに対して送出されるユーザーの属性について、送出する／しないを設定します<br>主にOrthrosと接続する際に初期設定として行います。また、SP側にて追加で属性が必要となった場合に設定変更することが可能です |

## 3\. Orthrosとの接続方法について

Orthrosと接続するには申込が必要となります  
申込方法については[こちら](/docs/SP管理者向けマニュアル/2%20Orthrosとの接続依頼方法について/README.md)からご確認をお願いします
