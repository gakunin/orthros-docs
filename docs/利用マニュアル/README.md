## **用語説明**

本マニュアルをはじめとしてOrthrosで使用される用語とその説明です

用語一覧

| 用語 | 説明 |
| --- | --- |
| OrthrosアカウントID | OrthrosアカウントのIDのことを指します。登録したメールアドレスと同一です。同じメールアドレスのアカウントを2つ以上作成することはできません |
| ローカルアカウント | Orthrosに登録されたパスワードにより認証されるアカウントを指します。  <br>Orthrosアカウントは3つの認証方式をサポートしています。  <br>1.ID（メールアドレス）とパスワードによる認証  <br>2.パスキーによる認証  <br>3.連携しているIdPによる認証  <br>ローカルアカウントは「1.ID（メールアドレス）とパスワードによる認証」及び「パスキー」で認証されるアカウントを指します。 |
| パスキー | パスキーは、FIDO2標準に基づくFIDO認証資格情報であり、ユーザーはデバイスのロック解除に使用するのと同じプロセス(生体認証、PIN、またはパターン)でアプリやWebサイトにサインインできます  <br>Orthrosアカウントにパスキーを登録することにより、パスワードの代わりに生体認証、PIN、パターン、セキュリティキー等で認証することが出来ます  <br>※Orthrosアカウントでパスキーを登録する方法は[こちら](#パスキーの新規登録) |
| 認証器 | パスキーにおける認証器（Authenticator）は、FIDO2標準に基づくセキュリティデバイスで、パスキーを保存するソフトウェアやハードウェアを指します  <br>※認証器の種類<br>・ローミング認証器（クロスプラットフォーム認証器）：USBキーなど、取り外し可能な認証器<br>・プラットフォーム認証器：PCやスマートフォンに内臓（固定）された取り外し不可な認証器 |
| クロスデバイス認証 | パスキーが登録されているデバイス（iPhone、iPad や Android デバイス）を認証器として使うことで登録がない別のデバイスからもパスキーでサービスにログインできるようにする方式（両デバイス間はBLEとQRコードを併用しダイレクトに連携します） |
| xID | xID(クロスアイディ)は、xID社が提供するサービスで、マイナンバーカードと連携することで、より手軽に本人確認・電子認証・電子署名、事業者へのマイナンバーの提供、マイナンバーカードによる公的個人認証を実施することができるデジタルIDアプリです  <br>Orthrosで、IAL2・AAL2のサポートを行う際に、xIDを利用します  <br>※xIDについては[こちら](https://xid.inc/home)  <br>※xIDアプリについては[こちら](https://xid.inc/application)  <br>※OrthrosアカウントでxIDと連携する方法は[こちら](#xIDとの連携登録) |
| IAL / IAL2 | IALは Identity Assurance Level の略で、発行されるアカウントが特定の個人に結びついているかを保証するレベルです。身元確認を行う文書その他で判断します。<br>IAL2はIALのレベル2で、高い保証度でアカウントと特定の個人の結びつきを保証するレベルです。通常、公的な身分証の提示等を要求します。 |
| AAL / AAL2 | AALは Authenticator Assurance Level の略で、アカウントを使ってログイン（認証）するときに、アカウントを持っている当人が確実にログイン作業を行っているかを保証するレベルです。<br>AAL2はAALのレベル2で、高い保証度で当人がログイン作業を行っていることを保証するレベルです。パスワードによる認証では不十分で、それに加えてに何かを所有していたり、生体に関する情報を提示することを要求します。 |
| マイページ | 利用中SPの一覧確認と利用中SPの同意確認設定変更ができるページ |
| 表示名 | マイページで表示させる名前  <br>※一部のSPに送信されます |
| メールアドレス | アカウントに登録するメールアドレス  <br>※一部のSPに送信されます |
| ePPN（eduPersonPrincipalName） | Orthros利用者の、学認において一意となる識別子  <br>※一部のSPに送信されます |
| 所属機関 | サインイン中のOrthrosアカウントが所属している機関（今後提供予定） |
| プロジェクト | サインイン中のOrthrosアカウントが所属しているプロジェクト（今後提供予定） |
| 利用中のSP | SPの利用状況と、Orthrosから各SPへの属性送信同意状況を表す項目です |
| IdPとの認証連携状況 | xIDとの連携状況を表す項目です  <br>今後他のIdPにも拡大予定です |

## **新規アカウント登録**

1. Orthros（[https://ui.orthros.gakunin.nii.ac.jp](https://ui.orthros.gakunin.nii.ac.jp)）へアクセスします。<br>
2. 「新規登録」ボタンを押下します。<br><img src="/images/画像21.png" width="40%" /><br>
3. メールアドレスを入力し、「認証コードを送信する」ボタンを押下します。<br><br>
<img src="/images/github_note_icon.png" width="11%" /><br>
認証コードの有効期間は10分です。<br><br>
<img src="/images/画像23.png" width="40%" /><br><br>
4. （メールアドレス検証では）入力したアドレスに届いた認証コードを入力し、「コードの確認」ボタンを押下します。<br><br>
<img src="/images/github_note_icon.png" width="11%" /><br>
「新しいコードを送信します」ボタンを押下すると、新しい認証コードが送信されます。<br>
<img src="/images/画像24.png" width="40%" /><br><br>
5. パスワード、日本語の姓名、英字の姓名を設定します。<br><br>
<img src="/images/github_note_icon.png" width="11%" /><br>
パスワードは半角英大文字、半角英小文字、数字、記号（スペースは除く）の4種類のうち3種類以上を使用した10文字以上のパスワードとしてください。<br>
パスワード、日本語の名、英字の名は入力必須となります。<br>
入力された姓名から表示名が自動生成されます。自動生成される表示名はこのようになります："姓 名"（日本語の場合）、"Givenname Familyname"（英字の場合）<br>
生成された表示名は後でマイページ内の設定で変更可能です。<br><br>
<img src="/images/画像25.png" width="40%" /><br><br>
6. Orthrosのマイページが表示されていれば、サインイン完了です。<br><img src="/images/image-2024-7-22_13-5-47.png" width="50%" />

## **Orthrosマイページへのサインイン**

>[!NOTE]
>マイページでできることは以下となります。<br>
>・表示名変更<br>
>・メールアドレス変更<br>
>・パスワードリセット<br>
>・パスキー管理（登録、名称変更、削除）<br>
>・利用中SPの一覧確認及び同意確認設定の変更<br>
>・xIDとの連携登録及び解除<br>
>・アカウント削除<br>
>・お問い合わせ<br>

1. Orthros（[https://ui.orthros.gakunin.nii.ac.jp](https://ui.orthros.gakunin.nii.ac.jp)）へアクセスします。
2. サインインを押下します。<br>
<img src="/images/image-2023-9-22_18-5-3-1.png" width="50%" /><br>
3. サインイン方法を選択します<br>
	  - パスキーで認証する場合<br>
（事前に登録が必要です。登録方法は[こちら](#パスキーの新規登録)）<br>
	1. パスキータブから、パスキーでサインインを押下します<br><img src="/images/image-2025-10-17_16-57-34-1.png" width="30%" /><br><br>
	2. 利用するパスキーを選択し、生体認証・PIN等で確認を実施します<br><br>
	<img src="/images/image-2025-10-17_17-4-14-1.png" width="20%" /><br><br>
	※パスキー選択時の画面は利用するデバイスやブラウザによってUIや表示される選択肢が異なります<br>
	※クロスデバイス認証を利用する際の注意点
		1. 登録操作を行っているデバイスおよびiPhone、iPad や Android デバイスの両方でbluetoothが利用可能な状態であること<br>
登録操作を行っているデバイスのbluetoothが利用不可の場合、選択肢として表示されません<br>
選択肢に表示されない場合は、bluetoothデバイスの状態を確認してください<br>
		2. 登録操作を行っているデバイスとiPad や Android デバイスがbluetoothで接続出来る状態であること<br>
デバイス同士が離れている等で接続出来ない状態の場合は、利用出来ません<br>
QRコードを読み取ったあと、処理が進まず「問題が発生しました」が一定時間後に表示される（タイムアウト）場合は、デバイス同士がbluetoothで接続出来る状態であることを確認してください


	  - メールアドレス/パスワードで認証する場合<br>
ID/パスワードタブから、メールアドレス/パスワードを入力し、サインインを押下します<br><img src="/images/image-2025-10-17_17-9-10-1.png" width="20%" />


	  - xIDで認証する場合（事前に登録が必要です。登録方法は[こちら](#xIDとの連携登録)）  
	1. xIDを押下します<br><img src="/images/image-2025-10-17_17-9-50-1.png" width="30%" />
	2. xIDの認証画面で、「xIDとの連携登録」にて連携した際のID（メールアドレス）で認証を行います<br><img src="/images/image-2025-1-28_23-51-24-1.png" width="15%" />

4. Orthrosのマイページが表示されていれば、サインイン完了です。<br><img src="/images/image-2024-7-22_13-6-56.png" width="50%" />

<img src="/images/画像29.png" width="50%" /><br><br>
2. Orthrosアカウント作成時に指定したメールアドレスを入力し、「認証コードを送信する」ボタンを押下します。<br><br>
<img src="/images/github_note_icon.png" width="11%" /><br>
認証コードの有効期間は10分です。<br><br>
<img src="/images/画像30.png" width="50%" /><br><br>
3. （メールアドレス検証では）入力したアドレスに届いた認証コードを入力し、「コードの確認」ボタンを押下します。<br><br>
<img src="/images/github_note_icon.png" width="11%" /><br>
「新しいコードを送信します」ボタンを押下すると、新しい認証コードが送信されます。<br><br>
<img src="/images/画像31.png" width="50%" /><br><br>
4. 認証コードの検証が完了しますので、「続行」ボタンを押下します。<br><br>
<img src="/images/github_note_icon.png" width="11%" /><br>
検証するメールアドレスを間違った場合は、「メールアドレスを修正します」ボタンを押下します。<br><br>
<img src="/images/画像32.png" width="50%" /><br><br>

<img src="/images/github_note_icon.png" width="11%" /><br>
パスワードは半角英大文字、半角英小文字、数字、記号（スペースは除く）の4種類のうち3種類以上を使用した10文字以上のパスワードとしてください。<br>
※変更前と同じパスワードは指定できません。<br><br>
<img src="/images/image-2025-10-23_9-32-11.png" width="30%" /><br><br>
6. パスワード変更後マイページへ遷移すれば、パスワードリセット完了となります。<br><img src="/images/image-2024-7-22_13-10-37.png" width="50%" />

## **SPでのログイン**

1. OrhtrosアカウントでSPへログインをするには、DSで「Orthros」を選択して、「選択」ボタンを押下します。<br>
※以下はGakuNin RDMでの例です。<br><img src="/images/利用者マニュアル-SPでログイン1.png" width="35%" />

2. サインイン方法を選択します<br>
	* パスキーで認証する場合
	* メールアドレス/パスワードで認証する場合
	* xIDで認証する場合<br>
※それぞれの方法については[こちら](#Orthrosマイページへのサインイン)の「サインイン方法を選択します」を参照

3. 追加認証<br>
	SPからAAL2を要求されている かつ AAL2を満たしてない場合、追加で認証を行います<br>
	※ID/パスワードで認証している場合は、追加での認証が必要となります<br>
	※パスキー、xIDで認証している場合は、追加での認証は不要となります<br>

	* パスキーを利用する場合<br>
パスキーを押下します<br>
<img src="/images/image-2025-10-20_10-33-49-1.png" width="30%" /><br><br>
利用するパスキーを選択し、生体認証・PIN等で確認を実施します<br>
<img src="/images/image-2025-10-20_10-38-4-1.png" width="20%" /><br>
※パスキー選択時の画面は利用するデバイスやブラウザによってUIや表示される選択肢が異なります<br><br>
	* xIDを利用する場合<br>
xIDを押下します<br>
<img src="/images/image-2025-10-20_10-39-53-1.png" width="30%" /><br><br>
xIDの認証画面で、「xIDとの連携登録」にて連携した際のID（メールアドレス）で認証を行います<br>
<img src="/images/image-2025-1-28_23-51-24-1.png" width="15%" /><br>

4. 次に属性提供同意画面が表示された場合はどちらかを選択して「続行」ボタンを押下します。後者を選択した場合は次回以降同じSPにログインする際に属性提供同意画面を表示しません。<br><img src="/images/image%20%2813%29.png" width="50%" /><br>

5. SPの画面に遷移するとログイン完了です。<br>
※以下はGakuNin RDMでの例です。<br><img src="/images/image%20%283%29.png" width="50%" />
